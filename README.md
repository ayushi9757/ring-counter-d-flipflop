🔁 Ring Counter using D Flip-Flop
🎯 Objective

Design and simulate a 4-bit ring counter using D flip-flops to demonstrate sequential shifting of a single logic ‘1’.

🧠 Working Principle

A ring counter circulates a single logic ‘1’ through flip-flops in a cyclic manner on each clock pulse.

🔄 Sequence

1000 → 0100 → 0010 → 0001 → 1000

⚙️ Tools Used
Xilinx Vivado
Verilog HDL
Digital Simulation Tool
📁 Files Included

testbench.v

📊 Result

The simulation confirms correct circular shifting of a single bit across four flip-flops.

🏁 Conclusion

The ring counter was successfully designed and simulated, demonstrating sequential digital logic behavior.
